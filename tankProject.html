Overview

The Tank Project is a robotics and control systems experiment where I designed and built a tracked vehicle capable of remote navigation and obstacle avoidance.

Key Features

  • Custom chassis with durable tracks
  • Arduino-based control system
  • Wireless communication for remote control
  • Camera integration for live feedback
